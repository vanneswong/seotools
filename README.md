# SEO查询浏览器插件

## 功能
点击插件图标，自动获取当前网站的域名，并跳转到站长工具SEO查询页面。

## 安装方法

### Chrome浏览器
1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本文件夹

### Edge浏览器
1. 打开 Edge，访问 `edge://extensions/`
2. 开启左下角"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择本文件夹

## 使用方法
1. 打开任意网站
2. 点击浏览器右上角的插件图标
3. 点击"查询SEO信息"按钮
4. 自动跳转到站长工具查询该域名的SEO信息

## 注意事项
- 图标文件需要为 PNG 格式，你可以将 SVG 文件转换为 PNG，或自行替换图标
- 部分特殊页面（如 chrome:// 页面）无法获取 URL
